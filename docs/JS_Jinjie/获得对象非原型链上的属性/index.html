获得对象非原型链上的属性

在 JavaScript 中，获取对象自身（非原型链上）的属性有几种不同的方法。具体使用哪一种，取决于你是否需要获取不可枚举属性以及 Symbol 类型的属性。

可枚举 的概念​

为了让你最直观地理解，我们可以打个比方：

想象你的对象是一个大仓库，里面的属性就是仓库里的物品。

可枚举（Enumerable）的属性：就像是摆在明面展柜上的商品。当有人来盘点库存（比如你使用 for...in 循环，或者 Object.keys()）时，这些商品会被一眼看到，并被清点进清单里。

不可枚举（Non-enumerable）的属性：就像是锁在抽屉或保险柜里的内部工具或机密文件。它们确实存在于仓库中（如果你知道保险柜的密码和位置，直接去拿 obj.secret 是一定能拿到的），但在常规的“盘点”过程中，它们会被刻意忽略，不会出现在库存清单上。

以下是几种常用的方法，按照获取属性的覆盖范围由小到大排列：

1. 获取自身的、可枚举的字符串属性：Object.keys()​

这是日常开发中最常用的方法。它只返回对象自身定义的、且 enumerable（可枚举）为 true 的字符串属性名数组。

const obj = { a: 1, b: 2 };
// 设置原型链上的属性
Object.setPrototypeOf(obj, { protoProp: 'I am on prototype' });

console.log(Object.keys(obj)); 
// 输出: ['a', 'b'] (排除了 protoProp)

2. 结合遍历使用：for...in + Object.hasOwn()​

for...in 循环会遍历对象自身及其原型链上的所有可枚举属性。结合 Object.hasOwn() （或老旧的 obj.hasOwnProperty()）可以过滤掉原型链上的属性。

注意： Object.hasOwn() 是 ES2022 引入的更安全的写法，用来替代 Object.prototype.hasOwnProperty.call()。

for (let key in obj) {
  if (Object.hasOwn(obj, key)) {
    console.log(key); // 只输出 'a', 'b'
  }
}

3. 获取自身所有的字符串属性（包含不可枚举）：Object.getOwnPropertyNames()​

如果你定义了一些不可枚举的属性（例如通过 Object.defineProperty 且 enumerable: false），Object.keys() 是拿不到的。这时你需要使用 Object.getOwnPropertyNames()。

const obj = { a: 1 };
Object.defineProperty(obj, 'hidden', {
  value: 'secret',
  enumerable: false
});
Object.setPrototypeOf(obj, { protoProp: 'I am on prototype' });

console.log(Object.keys(obj)); 
// 输出: ['a']

console.log(Object.getOwnPropertyNames(obj)); 
// 输出: ['a', 'hidden'] (包含了不可枚举属性，排除了原型属性)

4. 获取自身的 Symbol 属性：Object.getOwnPropertySymbols()​

如果对象的属性键是 Symbol 类型，上述方法都无法获取。你必须专门使用这个方法。

const sym = Symbol('mySymbol');
const obj = { a: 1, [sym]: 2 };

console.log(Object.getOwnPropertySymbols(obj)); 
// 输出: [ Symbol(mySymbol) ]

5. 获取自身所有的属性（终极方法）：Reflect.ownKeys()​

如果你想一次性拿到对象自身所有的属性键（不管是可枚举还是不可枚举，不管是字符串还是 Symbol），Reflect.ownKeys() 是最全面的选择。它的效果等同于 Object.getOwnPropertyNames(obj).concat(Object.getOwnPropertySymbols(obj))。

const sym = Symbol('mySymbol');
const obj = { a: 1, [sym]: 2 };
Object.defineProperty(obj, 'hidden', { value: 'secret', enumerable: false });
Object.setPrototypeOf(obj, { protoProp: 'I am on prototype' });

console.log(Reflect.ownKeys(obj)); 
// 输出: ['a', 'hidden', Symbol(mySymbol)]

方法对比总结​

方法自身可枚举自身不可枚举自身 Symbol原型链属性
Object.keys()✅❌❌❌
Object.getOwnPropertyNames()✅✅❌❌
Object.getOwnPropertySymbols()❌❌✅❌
Reflect.ownKeys()✅✅✅❌
for...in✅❌❌✅ (包含)
